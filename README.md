# Sorting algorithm visualizer 

A Java written GUI application to visualize sorting algorithms at work.  
You can select between 8 kinds of sorting algorithms:
* Bubble sort
* Cocktail sort
* Heap sort
* Insertion sort
* Merge sort
* Quick sort
* Radix sort
* Selection sort
